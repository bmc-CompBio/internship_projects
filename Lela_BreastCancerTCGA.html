Unsupervised Clustering of Breast Cancer Samples

Lela Makharadze

INTRODUCTION

Modern molecular cancer research is virtually inseparable from bioinformatical analysis and omic approaches are becoming common practice not only in academic but also diagnostic facilities [1].

Transcriptome and methylome analysis are some of the major tools for elucidating molecular profiles of different cancer types that in turn could lead to advancement in cancer treatment [2]. In order to facilitate these processes, a few publicly available databases collect a constantly increasing number of cancer samples. The Cancer Genome Atlas (TCGA) is one of such platforms that is frequently used for validation or reconsideration of common cancer classification, detection of biomarkers, assessing prognostic values of certain molecular profiles, etc. [3] [4].

Breast cancer is the most common type of cancer affecting women and the most frequent cause of cancer mortality in the same gender [5]. Defined by multiple criteria (such as immunohistochemistry, histopathology, molecular characteristics), all distinct subtypes can be still assigned to more general, hormone responsive and non-responsive groups which in turn determines the treatment course of a patient. Molecular classification has been subject to frequent refinement and more and more genes tend to be considered for diagnostic and treatment purposes [6]. Unsurprisingly, big data analysis has hugely contributed to these novelties in classification [7].

Having those uncertainties in mind, I set out to find out whether unsupervised clustering of RNAseq data could reveal distinct groups of samples from TCGA database and if so, which differentially expressed genes contribute most to the formation of these clusters. Additionally, I addressed methylation levels of the differentially expressed genes to test whether the epigenetic parameter could also reflect differences between samples.

Differential Gene Expression Analysis

Using limma package differential gene expression analysis (DEA) was conducted. The analysis allowed for identification of top up- and down-regulated genes depicted with the volcano plot.

#data modifications to add clusters to the expression matrix
colnames(ExpMatrixlogT) <- sub('\\|.*', '', colnames(ExpMatrixlogT)) # kept gene symbol and removed ENTREZ part from a gene name

# I tell the function about 2 groups and #exclude cluster IDs from calculation
groups <- as.factor(ExpMatrixlogUN[19948,])
designs <- model.matrix(~0 + groups, data=ExpMatrixlogUN[-19948,]) 


#Contrasts matrix
cms <- makeContrasts(difference = groups2-groups1, levels = designs)

fits <- lmFit(ExpMatrixlogUN[-19948,], designs)

vfits <- contrasts.fit(fits, contrasts=cms)

efits <- eBayes(vfits)

volcanoplot(efits, highlight = 7,  names = rownames(), hl.col = "blue")

Figure 3. Volcano plot of log2 fold-change (x-axis) versus -log10 FDR-corrected p-value (y-axis, representing the probability that the gene is differentially expressed). Genes with positive log2 fold are upregulated in one cluster compared to the other, while genes with negative log-fold change are considered downregulated.
